# 사또 (Satto)

> **내 사주 흐름으로 뽑는, 가장 개인적인 로또 번호**

사주(四柱)와 로또를 결합한 개인화 번호 추천 앱입니다. 생년월일·성별·태어난 시간을 입력하면 사주 오행 분석을 기반으로 로또 번호를 추천하고, 오늘의 복권운·운세 해석·행운 키워드를 함께 제공합니다.

## 주요 기능

### 사주 기반 번호 추천
- 생년월일을 천간(天干)/지지(地支)로 변환하여 오행 분석
- 년간·년지·월·일·오늘 날짜의 오행에 가중치 부여
- 성별에 따른 음양 보정 적용
- 1~45번 각 번호에 개인화된 가중치 산출

### 4가지 조합 타입
| 타입 | 설명 |
|------|------|
| 재물 기회형 | 재물 기운이 강하게 흐르는 과감한 조합 |
| 안정 상승형 | 무리하지 않는 안정감 속 균형 조합 |
| 직감 몰림형 | 직감이 빛나는 날의 감각적 조합 |
| 균형 흐름형 | 홀짝·번호대 균형을 맞춘 조합 |

### 오늘의 복권운
- 0~100점 애니메이션 바로 표시
- 대길 / 길 / 중길 / 평 / 소 등급 구분
- 오행별 행운 색상·키워드·추천 구매 시간대 제공

### 저장 & 히스토리
- 번호 저장 및 날짜별 기록 관리 (최대 50개)
- 자주 받은 번호 Top 7 통계
- 기록 삭제 기능

### 공유
- 네이티브 공유 API로 번호·복권운·운세 한줄 메시지 공유

## 시작하기

### 사전 준비
- Node.js 18 이상
- Expo Go 앱 (iOS / Android)

### 설치 및 실행

```bash
# 저장소 클론
git clone https://github.com/sc8556/satto-app.git
cd satto-app/satto-mobile

# 의존성 설치
npm install

# 개발 서버 시작
npm start
```

Expo Go 앱으로 QR 코드를 스캔하면 바로 실행됩니다.

### 플랫폼별 실행

```bash
npm run android   # Android
npm run ios       # iOS (macOS 필요)
npm run web       # 웹 브라우저
```

## 프로젝트 구조

```

├── app/
│   ├── _layout.tsx       # 루트 레이아웃 (딥네이비 테마)

│   ├── LottoCard.tsx        # 조합 카드
│   └── FortuneScore.tsx     # 복권운 점수 바
```
